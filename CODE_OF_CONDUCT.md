# Code of Conduct

OrbitTest is an open-source project. Everyone participating in issues, pull requests, discussions, and other project spaces is expected to help keep the community respectful and useful.

## Expected Behavior

- Be welcoming and patient.
- Assume good intent while still being clear.
- Give feedback on code and ideas, not on people.
- Accept constructive feedback gracefully.
- Respect different experience levels and backgrounds.

## Unacceptable Behavior

- Harassment, threats, insults, or personal attacks.
- Discriminatory language or behavior.
- Sexualized comments or unwanted attention.
- Publishing private information without permission.
- Repeated disruption of project discussions.

## Enforcement

Project maintainers may remove comments, close issues, reject contributions, or block participants when behavior harms the project community.

If you need to report a conduct concern, contact the project maintainer privately when a private contact is available. If no private contact is listed yet, open a minimal public issue asking for a private contact and do not include sensitive details.
